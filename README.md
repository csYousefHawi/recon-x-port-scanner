# recon-x-port-scanner
Simple Python-based Port Scanning Tool.
## Features
1 -  Host availability check (Ping)
2 - Common port scanning
3 - Service detection
4 - Colored terminal output
5 - Clean scan summary

## How It Works
The tool:
1. Checks if the target is reachable
2. Scans common ports
3. Identifies service names
4. Displays open and closed ports
